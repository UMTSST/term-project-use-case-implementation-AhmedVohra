
Use Case #1 Delete
Primary Actors: Administrator, Staff
Pre- Condition: The book should have a valid info in which contains it's name,Author name and ISBN.
Main Scenario
1. To Delete a book the administrator or staff has to first login.
2. After login the administrator or staff searches for the existing book.
3. If the book is already registered, Admin deletes that book.
4. If the book is not already registered then error will occur .
Alternate Scenarios
1. The administrator or staff fails to login.
2. Administrator or staff cannot search for a book.
